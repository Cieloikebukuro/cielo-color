使い方ガイド
目次
配合計算 ★ 追加分を計算 塗り分け カルテ スタイリスト
配合計算の使い方
1
スタイリストを選ぶ
配合計算ページ上部のボタンから自分の名前をタップ。
2
総量とオキシ設定を入力
オキシ含む：総量にオキシも含めて逆算　／　オキシ含まない：カラー剤のみで計算
3
カラー剤を入力
名前と比率を入力（例：A6＝1、A10＝2 → 1:2配合）。%ボタンでカラー剤合計比での指定も可能。
4
オキシを選ぶ
濃度（3% / 4.5% / 6%）と倍率を選択。比率を細かく指示するで自由比率入力も可。
5
アシスタント・メモを入力して保存
担当アシスタントを選んでメモを入力。「配合を計算する」→施術カテゴリ選択→「カルテに保存」。
★
追加分を計算
カルテをタップすると詳細が開きます。
「仕上がり総量を変更」欄に新しい総量を入力して「追加分を計算」を押すと、元の比率・オキシ倍率を保ったままグラム数が自動で逆算されます。

例：50g → 70gに変更したい
→ 20g分の追加量が各剤ごとに自動計算されます
塗り分け計算
「塗り分け」タブで部位ごとに別々の配合を計算できます。
パターン
全体 / 根元・毛先 / 根元・中間・毛先 / ブリーチ / インナー・グラデーション / ハイライト
ハイライト
ホイル枚数プリセット＋±で入力。オンカラーは任意追加。
施術カテゴリ（ワンカラー / ダブルカラー / ハイライト / グラデーション）を保存前に選択。カルテにバッジで表示されます。
カルテ・カレンダー
カレンダー
オレンジの点がある日は記録あり。タップするとその日のカルテが表示。
カルテ
スタイリストで絞り込み可。タップで詳細表示・追加分を計算。
スタイリスト管理
右上の名前ボタンをタップするとスタイリスト一覧が開きます。追加・削除もここから行えます。
